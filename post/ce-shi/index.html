常用库与资源站点收藏

平时收集的觉得好用的以及有意思的库与资源网

vue库

vue-3d-model

	一个展示三维模型的Vue组件，支持模型操作和模型点击事件，能自动缩放模型到合适大小并校正偏移，支持多种格式的模型。

yox

	一个兼容低版本浏览器的类 Vue 框架。

noahv

	NoahV是一个基于vue开发的前端开发框架，主要适用于监控运维、数据分析等场景，同时也适用于常见的后台管理系统

iview

	iView 是一套基于 Vue.js 的开源 UI 组件库

d2-admin

d2-admin 是一套基于 Vue.js 的开源 UI 组件库

element

element 是一套基于 Vue.js 的开源 UI 组件库

vuepress

Vue 官方出品的静态网站生成器

v-region

简洁强大的中国行政区划选择器，可选择 “省/直辖市”、“市”、“区/县”、“乡/镇/街道” 4 级行政区

legoflow

更轻、更强、开箱即用的 前端工作流客户端 vue

v-uploader

基于 Vue2 简洁易用、可批量、拖拽的文件上传插件。可以看一上这个作者的相关库

react库

Ant Motion

	阿里开源的项目，一套 React 框架动效解决方案，可以帮助开发者，更容易的在项目中使用动效。同时可以方便快捷地制作一个公司的介绍页

BizCharts

	阿里开源的项目，一套 React图片库，基于G2

react-image-process

	图片处理的 React 组件。支持压缩、裁剪、加水印、滤镜、获取主色调等功能

zhui

	这是一款国风的组件库。好用的组件库千千万，有趣的创意万里挑一

react-kawaii

	生成卡通头像

其他

ogl

 最小的 WebGL 框架

charts

	简单、响应迅速、现代SVG图表库 零依赖关系

china-area-data

	中国省市区数据

ECharts

	百度开源的图标库

SpinKit.css

	css动画loading库

papercss.css

手绘风格的css库

animate.css

css动画库

swiper

 用于实现页面的整屏滚动

normalize.css

 用于重置浏览器默认样式

css-loaders

 css动画loading库

zingtouch

 用于创建手势库

share

 一键分享到微博、QQ 空间、QQ 好友、微信、腾讯微博、豆瓣等社交网站的 JavaScript 项目

nodePPT

 这可能是迄今为止最好的网页版演示库

clipboard

实现了点击文本内容的 JavaScript 插件，优点：最新、极小、无任依赖、使用简单方便

atrament

js的画板

flv.js

B站开源的fly视频播放组件

screenfull.js

跨浏览器JavaScript全屏API

gifify

截取视频某一段转成GIF动画

xdomain

用于跨域的JS库，利用的是XHook(jquery)

slick

基于JQuery的轮播图库

DPlayer

可爱的弹幕视频播放器

APlayer

漂亮的 HTML5 音乐播放器

IDValidator

中国大陆个人身份证号验证 Chinese Mainland Personal ID Card Validation

hub.js

集线器，满足绝大部分情况事件驱动的情况，适合用于处理各种事件流

numeric-keyboard

手机数字键盘，包含一个纯键盘 UI 和输入框套件。有纯 JavaScript、React、Vue 三个版本

webster

node的爬虫框架

vtils

工具库

SVG-Skeleton

通过 SVG 元素去描述去骨骼图的占位元素

dayjs

Moment.js 的 2kB 轻量化方案，拥有同样强大的 API

hotkeys 

一个强健的 Javascript 库用于捕获键盘输入和输入的组合键。它没有依赖，压缩只有 3kb 左右

picojs 

200 行实现的面部识别库

axial3d 

轴动效果 axial3d

giojs 

基于 Three.js 的 Web 3D 地球数据可视化的开源组件库

rainbow 

语法高亮库

Chart.js 

基于 canvas 的可视化库。可用于构建简单的 H5 图表，满足基本的日常可视化需求

griffith 

知乎开源的视频播放器

xgplayer 

由字节跳动西瓜视频开源的带解析器、能节省流量的 HTML5 视频播放器

filepond 

一个 JavaScript 文件上传库。可以上传拖入的任何内容，具有体积小、上传快、方便的文件管理等特点，从而让用户享受‘丝滑’般的文件上传体验

tesseract.js 

可以完成 62 种语言 OCR 的 JS 库

微信

jquery-weui

可能是最好用 WeUI 版本

wechat-h5-boilerplate

为腾讯微信优化的H5动效模板，帮助你快速构建全屏滚动型H5页面。

wxapp-market

小程序营销组件。

wepy

小程序开发框架基于vue 腾讯

mpvue

小程序开发框架基于vue 美团

taro

小程序开发框架基于react 京东

BeautyWe

BeautyWe 是一套专注于微信小程序的企业级开发范式  Fundebug

wechaty

wechaty微信API

工具库

VeryNginx

VeryNginx 是一个功能强大而对人类友好的 Nginx 扩展程序.

standard

JavaScript 代码规范，自带 linter & 代码自动修正

zan-proxy

该项目集成了 HTTP 请求转发、模拟响应数据、自定义 DNS 解析等功能。使得可以本地代码调试线上页面，环境再也不是问题

gcoord

地理坐标系转换工具，支持WGS84/GCJ02/BD09等常用坐标系互转

whistle

是基于Node实现的跨平台抓包调试代理工具

vConsole

一个轻量、可拓展、针对手机网页的前端开发者调试面板。

Omi - 前端跨框架跨平台框架

基于 Web Components 并支持 IE8+(omio)，小程序(omip) 和 任意前端框架集成

squoosh

谷歌开源的图片压缩工具。在保证图片质量的情况下快速压缩图片，支持多种图片格式。6.63M 的图片压缩后为 2.92M，使用起来简单方便

在线工具

Speed-Line

loading
