常用库与资源站点收藏

平时收集的觉得好用的以及有意思的库与资源网

vue库

1. vue-3d-model

	一个展示三维模型的Vue组件，支持模型操作和模型点击事件，能自动缩放模型到合适大小并校正偏移，支持多种格式的模型。

2. yox

	一个兼容低版本浏览器的类 Vue 框架。

3. noahv

	NoahV是一个基于vue开发的前端开发框架，主要适用于监控运维、数据分析等场景，同时也适用于常见的后台管理系统

4. iview

	iView 是一套基于 Vue.js 的开源 UI 组件库

5. d2-admin

d2-admin 是一套基于 Vue.js 的开源 UI 组件库

6. element

element 是一套基于 Vue.js 的开源 UI 组件库

react库

1. Ant Motion

	阿里开源的项目，一套 React 框架动效解决方案，可以帮助开发者，更容易的在项目中使用动效。同时可以方便快捷地制作一个公司的介绍页

其他

1. ogl

 最小的 WebGL 框架

2. charts

	简单、响应迅速、现代SVG图表库 零依赖关系

3. china-area-data

	中国省市区数据

4. ECharts

	百度开源的图标库

5. SpinKit.css

	css动画loading库

6.papercss.css

手绘风格的css库

7.animate.css

css动画库

8.swiper

 用于实现页面的整屏滚动

9.normalize.css

 用于重置浏览器默认样式

10.css-loaders

 css动画loading库

11.zingtouch

 用于创建手势库

12.share

 一键分享到微博、QQ 空间、QQ 好友、微信、腾讯微博、豆瓣等社交网站的 JavaScript 项目

13.nodePPT

 这可能是迄今为止最好的网页版演示库

14.clipboard

实现了点击文本内容的 JavaScript 插件，优点：最新、极小、无任依赖、使用简单方便

15.atrament

js的画板

16.flv.js

B站开源的fly视频播放组件

17.screenfull.js

跨浏览器JavaScript全屏API

18.gifify

截取视频某一段转成GIF动画

19.xdomain

用于跨域的JS库，利用的是XHook(jquery)

20.slick

基于JQuery的轮播图库

21.DPlayer

可爱的弹幕视频播放器

22.APlayer

漂亮的 HTML5 音乐播放器

23.IDValidator

中国大陆个人身份证号验证 Chinese Mainland Personal ID Card Validation

24.hub.js

集线器，满足绝大部分情况事件驱动的情况，适合用于处理各种事件流

25.numeric-keyboard

手机数字键盘，包含一个纯键盘 UI 和输入框套件。有纯 JavaScript、React、Vue 三个版本

微信

1. jquery-weui

可能是最好用 WeUI 版本

2. wechat-h5-boilerplate

为腾讯微信优化的H5动效模板，帮助你快速构建全屏滚动型H5页面。

3. wxapp-market

小程序营销组件。

4. wepy

小程序开发框架基于vue

工具库

1. VeryNginx

VeryNginx 是一个功能强大而对人类友好的 Nginx 扩展程序.

2. standard

JavaScript 代码规范，自带 linter & 代码自动修正

在线工具

1. 免费且易于使用的在线PDF工具

2. 给你提供许多种颜色搭配，每种都是四种颜色

3. 蚂蚁金服设计平台

4 在线IP地址

IP 查询网站，支持 IPv6 可获取来源国家、网络商等信息

5. Speed-Line

快速画出有冲击力的线条

6. loading

这个网站可以帮你生成现成的加载动画
